# Nix

## 安装

> <https://zero-to-nix.com/start/install/>

```bash
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install
```

查看是否安装完成（需要打开新的 shell 窗口）：

```bash
nix --version
```
